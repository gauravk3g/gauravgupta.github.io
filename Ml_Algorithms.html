Machine Learning Algorithms and AI Domains

Portfolio Artifact Demonstrating Understanding of AI / ML Models

Introduction

Artificial Intelligence and Machine Learning have become essential technologies in modern computing. Many systems today rely on machine learning algorithms to analyze data, identify patterns, and make predictions. These algorithms are used in a wide range of domains including finance, healthcare, marketing, language processing, and computer vision.

This portfolio artifact presents an overview of several commonly used machine learning algorithms and explains how they relate to major AI domains such as tabular data analysis, computer vision, natural language processing, and generative AI.

Description

Machine learning algorithms can generally be categorized based on how they learn from data. Supervised learning algorithms learn from labeled datasets and are commonly used for prediction and classification tasks. Unsupervised learning algorithms work with unlabeled data and focus on discovering patterns or groupings within the dataset.

In addition to learning style, algorithms are often associated with particular application domains. For example, decision trees and random forests are commonly used with structured tabular datasets, while convolutional neural networks are widely used for computer vision tasks. Transformer-based models are frequently used in natural language processing applications, and generative models such as GANs and diffusion models are used to create new content.

Objective

  • Identify and categorize important machine learning algorithms.
  • Understand how different algorithms relate to AI domains.
  • Provide simple explanations of how each algorithm works.
  • Illustrate real-world examples of how these algorithms are applied.
  • Create a visual and organized artifact suitable for a professional portfolio.

Process

Decision Tree

Supervised Tabular Data

Splits data into branches based on feature values to reach a decision.

Random Forest

Supervised Tabular Data

Combines multiple decision trees to improve prediction accuracy.

Linear Regression

Supervised

Finds relationships between variables and predicts continuous values.

Support Vector Machine

Supervised

Separates data into classes using an optimal boundary.

K-Means Clustering

Unsupervised

Groups similar data points into clusters.

Principal Component Analysis

Unsupervised

Reduces the number of features in a dataset while retaining key information.

Convolutional Neural Network

Deep Learning Computer Vision

Analyzes images by detecting patterns like shapes and textures.

Transformer Models

Deep Learning NLP

Uses attention mechanisms to understand language context.

Generative Adversarial Networks

Generative AI

Two neural networks compete to generate realistic synthetic data.

Diffusion Models

Generative AI

Gradually removes noise to generate new images.
